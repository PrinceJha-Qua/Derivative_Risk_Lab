# Derivative Lab

Understanding derivatives by building them from scratch.

Most resources teach derivatives through formulas.

This project takes the opposite approach:

- Visualize payoff geometry
- Build option strategies
- Construct binomial pricing models
- Derive replication-based pricing intuition
- Progress toward Black–Scholes from first principles

## Roadmap

✅ Phase 1 - Payoff Geometry

- Long Call
- Long Put
- Short Call
- Short Put
- Straddles
- Strangles
- Bull Spreads
- Protective Puts
- Covered Calls

✅ Phase 2 - Binomial Pricing

- One-step model
- Multi-step tree
- Risk-neutral probabilities
- Backward induction
- Replicating portfolio

⬜ Phase 3 - Black–Scholes

⬜ Phase 4 - Greeks

⬜ Phase 5 - Monte Carlo
